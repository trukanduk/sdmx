      Обработанный xml-файл

      По сути своей этот объект - обертка для массива, возвращаемого функцией xml_parse_into_struct() Имеет свой внутренний указатель на текущий тег.

      package sdmx\parseUtils
      version 1.2

      Текущий элемент

      Curr(mixed $attr, mixed $default) : mixed

      Возвращает ссылку на текущий элемент. Если параметр $attr - строка, то возвращается значение свойства $attr текущего элемента

      Parameters

      $attr

      mixed

      если false (по умолчанию), то возвращается весь текущий элемент, если строка, то свойство текущего элемента (или $default)

      $default

      mixed

      возвращается в случае отсутствия элемента или отсутствия свойства

      Returns

      mixedТекущий элемент массива, значение свойства или $default, если его нет (свойства или самого объекта)

      Пропуск тегов

      Skip(string $name, mixed $type, mixed $default) : mixed

      Пропускает все теги, включая тег с именем $name и типом $type (если $type установлен как false, он игнорируется) Если пропущен весь массив, вернётся $default

      Parameters

      $name

      string

      имя тега, который надо пропустить

      $type

      mixed

      тип тега, если установлен в false, игнорируется

      $default

      mixed

      значение, которое возвращается в случае, если пропущен весь массив

      Returns

      mixedтег после тега с именем $name и типом $type (XmlTagInArray) или $default, если пропущен весь массив

      Пропуск тегов до нужного

      SkipTo(string $name, mixed $type, mixed $default) : mixed

      Пропускает все теги до тега с именем $name и типом $type (если $type === false, то тип не учитывается), возвращает $default, если пропущен весь массив

      Parameters

      $name

      string

      имя тега, до которого надо всё пропустить

      $type

      mixed

      тип искомого тега, при false типы тегов игнорируются

      $default

      mixed

      возвращается в случае пропуска всего массива

      Returns

      mixedискомый тег (XmlTagInArray) или $default в случае пропуска всего массива
