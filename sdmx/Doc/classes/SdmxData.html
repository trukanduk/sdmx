      Получение аттрибута заголовка

      GetHeaderAttr(string $attr, mixed $default) : mixed

      Возвращает значение аттрибута заголовка с именем $attr. В случае отсутствия вернётся $default Имя аттрибута приводится к нижнему регистру

      Parameters

      $attr

      string

      имя аттрибута. Приводится к нижнему регистру (т.е. 'key' и 'KEY' -- одинаковые ключи)

      $default

      mixed

      значение, которое вернётся в случае отсутствия аттрибута с именем $attr

      Returns

      mixedзначение аттрибута заголовка (т.е. string) или $default в случае отсутствия аттрибута заголовка

      $description : string[]

      По сути ещё один header. Массив хранится в виде ['<имя описания>' => '<значение>'], причём имя регистронезависимо (всегда приводится к нижнему регистру) Вообще, в теге description может быть несколько тегов indicator, однако здесь считается, что он может быть только один.       Массив заголовков файла в виде ['<имя>' => '<значение>']. В случае тега Sender имя будет выглядеть как 'sender::name' Все ключи приводятся к нижнему регистру.
