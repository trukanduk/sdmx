      Codelist

      Некий список неких элементов :) В Xml такой пример:

      <structure:Codelist id="NAME">
  <structure:Name>VALUE</structure:Name>
  <structure:Code value="CONSTANT_NAME_0">
      <structure:Description>CONSTANT_DESCRIPTION_0</structure:Description>
  </structure:Code>
  <structure:Code value="CONSTANT_NAME_1">
      <structure:Description>CONSTANT_DESCRIPTION_1</structure:Description>
  </structure:Code>
</structure:Codelist>

      Объект содержит в себе массив объектов SdmxCodelistInstance

      Объект неизменяем вне самого себя, поэтому можно безболезненно передавать его куда-либо

      package sdmx
      version 1.1

      $defaultInstanceName : string

      В некоторых случаях оказывается, что должен быть элемент списка по умолчанию. В этом поле содержится имя этого элемента в массиве элементов Он может быть установлен из функции self::SetDefaultInstanceName() или если он будет будет первым в массиве (см. функцию self::AddCInstance())

       

      $description : string

      Имя списка на человеческом языке. в файле имеет название "name". Свойство read-only

       

      $instances : \SdmxCodelistInstance[]

      Массив элементов списка -- экземпляров SdmxCoelistInstance Хранятся в виде

      array('<имя элемента>' => SdmxCodelistInstance('<имя элемента>', '<описание элемента>'), ...)

      Имя элемента не зависит от регистра. Каждый из них неизменяем после создания, поэтому можно не копировать их при передаче

       

      $name : string

      Имя списка, в xml называется идентификатором (id). Свойство read-only.
