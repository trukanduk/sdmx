SetCodelistFromAray( \SdmxCodelist[]   $codelists, ) : bool

Description

Установка нужного списка значений

Функция нужна для того, чтобы установить правильный список значений. Если в $this->codelist хранится имя списка, то он будет найден в массиве и установлен. Предполагается, что массив такого же вида содержится в SdmxData Также происходит поиск в стандартных списках (см. SdmxPredefinedCodelists) Если $this->codelist и так было списком, то ничего не изменится (возвращено будет true)


Arguments

Name Type Description Default
$codelists \SdmxCodelist[]

массив списков в следующей форме: ['<имя списка>' => SdmxCodelist()]

Return value

Type Description
bool <var>true</var>, если в результате в <var>$this->codelist</var> оказался объект SdmxCodelist, <var>false</var> при неудаче
